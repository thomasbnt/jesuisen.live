# [Jesuisen.live](https://jesuisen.live)

> **Note**
> 
> jesuisen.live means "I'm live". Show us with a simple link your Twitch Stream with beautiful informations in the embed SEO.

Will  Build with :
- Fastify
- Twitch API
- Awesome devs

## Development

Install dependencies : 
```bash
yarn
```

Run the server : 
```bash
yarn serve
```
